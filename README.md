# Alexa: A Voice Assistant

A Python-based voice assistant inspired by Amazon Alexa.  
It listens to your voice, understands commands, and responds with speech.

## ✨ Features
- 🎤 Speech recognition (convert voice to text)
- 🔊 Text-to-speech responses
- 🎶 Play music from your library
- 🌐 Open websites in browser
- ⏰ Tell current time
- 🧠 Fuzzy matching for flexible commands

## 📦 Requirements
All dependencies are listed in `requirements.txt`.  
Install them using:
```bash
pip install -r requirements.txt
